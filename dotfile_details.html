Maintainer: suckless_fan | Updated: 2023-10-12 | License: MIT
#

README.md

A minimalist dwm configuration focusing on core functionality. This build adheres strictly to the suckless philosophy of "software that doesn't suck." It aims to be as small and efficient as possible while providing necessary modern conveniences.

Included patches for version 6.2:

  • - vanitygaps: For aesthetic window spacing.
  • - statuscolors: Support for colored text in the status bar.
  • - dwmbar: A clean, modular status bar script.

Installation

$ git clone https://git.suckless.org/dwm $ cd dwm $ cp config.def.h config.h $ make && sudo make install
